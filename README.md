# NeulBom
늘봄 실버타운 & 요양원

# 프로젝트 소개
- 입주자들의 건강, 삶의 질, 사회 참여 등을 증진하기 위한 실버타운과 요양원을 운영
하고, 다양한 서비스와 활동을 제공하기 위해 필요한 기능을 구상한다.
- 이를 위해 효율적인 운영과 관리를 위한 관리자용 그룹웨어와 실버타운 통합 웹 페이지를 구현한다.

# 개발기간
- 2023-05-25 ~ 2023-06-12
